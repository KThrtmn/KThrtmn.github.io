Design Principles

White Space

Walmart


Walmart

Walmart used the White Space design principle by effectively spacing out their layout. Walmarts layout is not cramped which allows for a cleaner and more effective design. This type of design allows for users to convey information faster and easier.

Alignment

Goulet Pen


Goulet Pen

Goulet Pen uses alignmet throughout their website. Items for sale are listed with in a set boundry. On the home page their sale listed is alignmed with the cart and the three dashes. Alignment helps with ease of use.

Repetition

Target


Target

Target uses repetition by repeting one font style throughout there web sight. This makes the web page easy to read and causes your eye to flow throughout whole web page.
